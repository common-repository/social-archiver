== Description ==

Social Archiver let's you backup your social life into WordPress in an easy way.

Social Archiver is supporting Twitter for now, Instagram & Tumblr support are planned in the next releases.

== Changelog ==

The changelog can be found [here](https://github.com/gallib/social-archiver/wiki/Changelog)

== Installation ==

To install Social Archiver

1. Upload the plugin files to the `/wp-content/plugins/social-archiver` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings -> Social Archiver screen to configure the plugin
1. Enjoy!
